# SenYumBook

SenyumBook adalah aplikasi digital sederhana untuk membantu pengguna mencatat, mengelola, dan melacak koleksi buku secara efisien & menyenangkan.

## Visi
Mempermudah proses pencatatan, pelacakan, dan manajemen perpustakaan pribadi atau kecil, sekaligus mendorong minat membaca melalui pengalaman pengguna yang ramah dan modern.

## Fitur Utama
- Autentikasi user
- Dashboard koleksi buku
- Pencarian & filter buku
- Input & edit data buku
- Statistik & laporan bacaan
- Tema gelap/terang
- Sinkronisasi cloud (opsional)

## Rencana Pengembangan
1. Inisialisasi project dan setup repository
2. Pengembangan fitur core (CRUD buku, daftar koleksi)
3. Penambahan autentikasi user
4. Implementasi dashboard statistik
5. Optimasi UI/UX (dark mode, responsive)
6. Integrasi fitur pencarian & filter
7. Penambahan fitur laporan/sinkronisasi/ekspor data

## Cara Berkontribusi
Silakan fork repo ini, buat branch sesuai fitur/bug, lakukan perubahan, lalu ajukan pull request.

## Teknologi
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```
